# AnyBrain

AnyBrain is a powerful desktop AI workspace manager that allows you to manage and interact with multiple AI platforms in one place.

## Features

- **Multi-Tab Interface**: Switch seamlessly between different AI platforms like ChatGPT, Claude, Gemini, and more.
- **Preset Platforms**: Quickly add popular AI services with built-in presets.
- **Custom URLs**: Add any AI or web platform by providing its URL.
- **Native Experience**: Built as a lightweight desktop application using Tauri.

## Technology Stack

- **Framework**: [Tauri 2](https://v2.tauri.app/)
- **Frontend**: React 19 + TypeScript + Vite
- **Icons**: Lucide React
- **Backend**: Rust

## Development

```bash
# Install dependencies
npm install

# Run in development mode
npm run tauri dev

# Build for production
npm run tauri build
```
